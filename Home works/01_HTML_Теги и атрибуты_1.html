Из множества существующих языков программирования Python &mdash один из самых легких
в изучении. Он создан в конце 80-х годов ХХ века, но с тех пор он сильно изменился.
Хотя Python входит в состав в состав большинства дистрибутивов Linux, на него часто не
обращают внимания при выюоре языка для изучения. В этой статье мы рассмотрим
создание консольных приложений, а в следующей &mdash программирование графических
интерфейсов (GUI). Давайте создадим простую программу прямо сейчас.

Наша первая программа

Наберём код в текстовом редакторе (нпример, в gedit). Потом мы разберемся, что делает
каждая строка и пойдём дальше.

Введите эти 4 строки:

#!/usr/bin/env python
print 'Hello. I am a python program.'
name = raw_input ("What is your name?")
print "Hello there, " + name + "!"

Это все. Сохраните файл как hello.ру в любом месте. Я предлагаю сохранить его в папке python_examples в домашнем каталоге. Этот пример показывает, как часто просто написать программу на Python. Для запуска программы нужно сделать её файл исполняемым. Для этого введите в терминале

C +x hello.ру

в папке, в в которой вы сохранили программу. Теперь запустите её.

greg@earth:~/python_examples$ . /hello.ру
Hello. I am a python program.
What is youe name? Fred Burphel
Hello there, Fred Burphel!
greg@earth:~/python_examples$

Пример цикла For

Теперь давайте рассмотрим пример программирования цикла. Запустите текстовый редактор и наберите следующую программу:

#!/usr/bin/env python
fro cntr in range (0,10) :
print cntr

Не забудьте поставить символ табуляции перед строкой "print cntr"/ Это важно. в Python для указывания на блоки кода используются не круглые ( и фигурные { скобки как в других языках программирования, а отступы.
